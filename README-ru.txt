Техническая дема игры Battle Tanks. 
Мини-руководство бета-тестера. :)

Инсталляция:
Распакуйте архив с игрой в любую папку. (например C:\Games\Btanks)

Игра требует наличие в системе библиотек openal. Для их установки запустите openalweax.exe, находящийся в директории с игрой. 

Инсталлятор/настройки *пока* не включены в дистрибутив, поэтому для того чтобы поменять режимы работы необходимо указание ключей командной строки. (список ниже)

Игра может работать в двух режимах :
1. Использование GL-рендерера. Для этого Вам, возможно, понадобится оригинальные драйвера для Вашей видеокарты. Иначе Вы, возможно, столкнётесь с чрезвычайно низкой производительностью графического движка. 
2. Использование софтварного рендерера. Работает практически во всех случаях. 

Ключи командной строки: 
--no-gl     выключает GL-рендерер (используется если в системе нет 3d-ускорителя)
--vsync     включает vsync (используется если в GL-режиме появляется мерцание части экрана или так называемый tearing effect)
--fs        полноэкранный режим (в будущем будет включено по умолчанию)

размер окна/разрешение экрана контролируется следующими ключами: 
-0 640x480
-1 800x600 (по умолчанию)
-2 1024x768
-3 1152x864
-4 1280x1024

--connect=имя/айпи-адрес  подключается к компьютеру в качестве клиента многопользовательской игры. 
--no-sound                выключает все звуки и музыку в игре.

Установки игры: 

В технической демонстрационной версии доступен файл конфигурации в котором можно настраивать различные параметры движка и игры. В большинстве случаев название говорит само за себя, но на несколько ключей я хотел бы обратить ваше внимание:

"player.control-method" : используемый метод управления. доступные значения : 
"keys":    курсорные клавиши + левый ctrl, левый alt
"keys-1":  rdfg + q + a.
"keys-2":  курсорные клавиши + правый shift, правый ctrl. 
(keys-1/2 используются в режиме разделения экрана между двумя игроками)

доступно управление геймпадами/джойстиками: "joy-1" и "joy-2" для первого и второго джойстика соответственно. 

Логи игры ведутся в файлике stderr.txt.
